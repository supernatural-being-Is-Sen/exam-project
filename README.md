# coolyusen-exam

#### 介绍
考试系统

#### 软件架构
spring-boot
微服务架构：spring-cloud
消息队列：rabbitmq
缓存数据库：redis
工具包：guava
分布式session:spring session
持久层框架：mybatis-plus
连接池：druid
前端:vue,bootstrap,jquery



#### 安装教程

将项目配置的ip进行修改
需要配置redis,mysql,rabbitmq

#### 使用说明

先启动注册中心,然后启动各个模块的provider和constomer
